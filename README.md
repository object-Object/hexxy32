# hexxy32

A RISC-V (`rv32i`) computer implemented in Hex Casting. Heavily WIP.

![](https://i.imgur.com/0LUewBC.png)

## Requirements

* Hex Casting 0.10.3
* Hexal
* Hex Gloop
